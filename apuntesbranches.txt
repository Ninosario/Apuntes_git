Para agregar una rama nueva se puede emplear el siguiente comando.

	git branch (nombre de la rama)

Y puedes eliminarla con 

	git branch (nombre de la rama) -D

	git branch

Para cambiarte de rama 

	git checkout (nombre de la rama)
